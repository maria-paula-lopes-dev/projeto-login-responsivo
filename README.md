# 🔐 Login Responsivo

Projeto de estudo desenvolvido com **HTML5 e CSS3**, focado na criação de uma **interface de login responsiva**, adaptável para diferentes tamanhos de tela por meio de **media queries e Flexbox**.

## 🔗 Demonstração
👉 Acesse o projeto online:  
https://maria-paula-lopes-dev.github.io/projeto-login-responsivo/

---

## 📌 Sobre o projeto

Este projeto apresenta uma **interface de autenticação simples e responsiva**, com layout centralizado e adaptado para **mobile, tablet e desktop**.

O layout utiliza **imagens como background**, que mudam de comportamento conforme o tamanho da tela, além de um formulário de login estilizado com ícones e organização visual focada na usabilidade.

---

## 🎯 Objetivos

- Criar uma interface de login responsiva  
- Praticar estruturação de **formulários em HTML**  
- Trabalhar **layout com Flexbox**  
- Aplicar **media queries para diferentes breakpoints**  
- Utilizar **imagens como background no layout**  

---

## 🧠 Conceitos aplicados

- Estruturação de formulários com **HTML5**
- Layout centralizado com **Flexbox**
- Uso de **ícones em campos de formulário**
- Background com **imagens adaptadas por breakpoint**
- Responsividade com **media queries**
- Organização de estilos em arquivos CSS separados

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- Flexbox  
- Media Queries  
- Google Material Symbols

## 📖 Aprendizados

Durante o desenvolvimento deste projeto foram reforçados conhecimentos como:

- Criação de interfaces responsivas
